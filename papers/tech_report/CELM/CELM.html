Algorithm

        CELM, a modified extreme learning machine, is an algorithm, using Sigmoidal function as activation function, which trains a single-hidden layer forward neural network (SFNN).

Matlab codes

        CELM Demo for Diabetes Version 1.0

Algorithm features

  • CELM trains a single hidden layer forward neural network with Sigmoidal function as activation function

  • Keeps the column rank of the hidden layer output matrix full

References

  • Z. Chen, H. Zhu, Y. G. Wang. A modified extreme learning machine with sigmoidal activation functions. Neural Comput. Appl., 22(3-4):541--550, 2013.
