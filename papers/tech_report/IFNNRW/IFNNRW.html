Iterative algorithm for feedforward neural networks with random weights (IFNNRW)

Technical report

  • IFNNRW trains a single hidden layer forward neural network with Sigmoidal function as activation function

  • Using rank-1 and rank-2 methods to obtain the optimal estimate for the output weights when column and row ranks of the hidden layer output matrix are not full
