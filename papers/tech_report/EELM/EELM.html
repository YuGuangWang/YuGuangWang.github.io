Algorithm

        EELM, a modified extreme learning machine, is an algorithm, using Gaussian function as activation function, which trains a single-hidden layer forward neural network (SFNN).

Matlab codes

        EELM Demo for Diabetes Version 1.0

Algorithm features

  • EELM trains a single hidden layer forward neural network with Sigmoidal function as activation function

  • Keeps the column rank of the hidden layer output matrix full
